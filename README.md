# Real Estate Application

Welcome to the Real Estate Application! This project is a full-stack web application where users can register, log in, post real estate properties, perform all CRUD operations, add properties to their favorite lists, chat with other users in real-time, and manage their profiles. The backend is built using Node.js, and the frontend is developed with React. Real-time communication is handled using Socket.io.

## Features

- **User Authentication**: Register and log in using secure authentication.
- **Property Listings**: Create, read, update, and delete real estate property listings.
- **Favorites**: Add properties to your favorite list to show interest.
- **Real-Time Chat**: Communicate with other users who have posted properties using a real-time chat application.
- **Profile Management**: Update your profile by uploading profile pictures and changing your personal data.
- **Search and Filter**: Search and filter properties based on various criteria.

## Technologies Used

- **Backend**: Node.js, Express.js, MongoDB, Mongoose
- **Frontend**: React, Axios
- **Real-Time Communication**: Socket.io
- **Authentication**: JWT (JSON Web Tokens), bcrypt
- **File Uploads**: Multer
- **Deployment**: Heroku

### Prerequisites

- Node.js (v14 or higher)
- npm (v6 or higher) or yarn
